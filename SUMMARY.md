## 🚀 Быстрый старт

```bash
# 1. Установка (выберите один из способов)

# Способ A: Через npm (рекомендуется)
npm install -g popapi

# Способ B: Автоматический скрипт
curl -sSL https://raw.githubusercontent.com/yourusername/popapi/main/install.sh | bash

# Способ В: Локально (для разработки)
git clone https://github.com/yourusername/popapi.git
cd popapi && npm install


# 2. Настройка
popapi setup
# Введите: GitHub token, username, default repo


# 3. Использование
popapi push ./my-files my-repo
popapi push ./build my-repo main
popapi push ./src my-repo develop
```

## 🔧 Подготовка к публикации на npm

1. Обновите данные в package.json:
```json
{
  "repository": {
    "url": "https://github.com/yourusername/popapi"
  },
  "author": "Your Name <your@email.com>"
}
```

2. Инициализируйте Git:
```bash
git init
git add .
git commit -m "Initial commit: popApi v1.0.0"
git remote add origin https://github.com/yourusername/popapi.git
git branch -M main
git push -u origin main
```

3. Опубликуйте на npm:
```bash
npm login
npm publish
```
